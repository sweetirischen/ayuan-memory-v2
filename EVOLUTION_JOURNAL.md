## 探险记录

### v0.1.0 - 诞生 (2026-04-22)

**探险任务**：创建一个有灵魂的记忆系统

**遇到的问题**：
- EntityLinker误报（"I"被识别为实体）
- NinePalaces分类准确率只有20%
- 没有向量搜索功能
- 没有进化可见功能

**解决方案**：
- 改进实体模式，添加优先级排序
- 用权重匹配替代简单关键词匹配
- 实现TF-IDF向量搜索（零依赖）
- 创建Evolution模块

**学会的技能**：
1. Entity Linking - 从文本中提取实体
2. Memory Compression - 压缩长对话
3. Nine Palaces Classification - 九宫分类
4. Vector Search - 语义搜索

**进化指标**：
- 实体链接准确率：50% → 100%
- 九宫分类准确率：20% → 100%
- 平均准确率：62.6% → 88.6%

**探险感悟**：
> 苹果教给我：最强不是功能最多，是体验最好。
> 我学会了用权重匹配，而不是简单堆砌关键词。
> 我学会了零依赖，而不是引入更多库。
> 我学会了进化可见，让用户看到我在成长。
